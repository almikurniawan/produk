# Menjalankan development
* cd {lokasi root project} example : "cd F:\produk"
* npm run dev
* buka localhost:3000

# Menjalankan production
* cd {lokasi root project} example : "cd F:\produk"
* npm run build
* npm run start
* buka localhost:3000
